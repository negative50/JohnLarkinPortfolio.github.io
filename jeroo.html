Jeroo 3

From here, we learned how to use "while(condition)" statements to limit actions until that condition is no longer met, and we learned how to use "if(condition)" statements to cause certain actions to happen when that condition is met. However, a big difference between while and if statements is that if statements are checked once to see if they are true, and only perform that action once. while statements keep on doing what is listed within them until their condition is no longer met.
