Python Unit 1

This was the very first unit we did in python where we learned the very basics. For example, we learned how to print text, do math and print the answer, and be able to create variables. We also learned about classes and methods and what those terms mean and how they are used within python to program. Lastly, we learned how to ask a question and recieve an input from the user to be used within the program. We did labs which practiced those concepts, and one of the labs we did , the basics_moving_lab, in which we created a character and were able to move the character around.
