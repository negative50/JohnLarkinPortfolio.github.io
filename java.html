Java

In this unit we messed around with the program, Eclipse Java, to learn the very basics of Java. For example, we learned about Java syntax, output, input, and if-else strings. We would then program a story with choices that incoporates all of these concepts. Syntax and output was used to produce the text that would appear after the choices are made, and to give the player the choices they could make. It also separated the if-else strings. The input used functions such as "Scanner keyboard = newScanner(System.in);" to allow the player to make choices which would in turn trigger other if-else strings to dislay text and create more options. If-else strings were used to cause the decisions to send the player down different paths that led to other if-else strings and so on and so forth.
