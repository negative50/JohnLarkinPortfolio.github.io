Final Project!

This is my final project for this class. This is a project created using python and in it, you fight hordes of monsters and rack up points while traversing through beautifully drawn backgrounds. The enemies are randomly generated using a list of "costumes" and random.randint[a,b] to make sure that every wave you face is probably unique and not the same as the previous wave. The farther you get into the game the harder it is to kill the enemies! In this project I used class files, importated from class files, incorporated a lot of variables, lists, and more to create this project.
